# SmartCalc Python

A modern calculator desktop application built using Python and Tkinter.

## Features
- Clean two-color UI
- Basic arithmetic operations
- Backspace and clear buttons
- Real-time clock
- Responsive desktop layout

## Tech Used
- Python
- Tkinter

## Why I Built This
As a first-year Mechanical Engineering student, I wanted to explore programming and GUI development by building a practical desktop application.
